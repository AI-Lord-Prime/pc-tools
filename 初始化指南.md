# 查机工具箱 - 项目初始化指南

## 一、环境依赖

### 1. Node.js
- **版本要求**: >= 18.0.0
- **安装地址**: https://nodejs.org/

### 2. Rust
- **版本要求**: >= 1.70.0
- **安装地址**: https://www.rust-lang.org/tools/install
- **Windows 安装**: 运行 `rustup-init.exe`，选择默认配置

### 3. Tauri CLI
```bash
# 全局安装 Tauri CLI
npm install -g @tauri-apps/cli
```

### 4. Visual Studio Build Tools (Windows)
- 需要安装 C++ 构建工具
- 安装地址: https://visualstudio.microsoft.com/zh-hans/downloads/
- 勾选 "Desktop development with C++" 工作负载

## 二、项目初始化

### 1. 克隆项目（如果是首次获取）
```bash
git clone <仓库地址>
cd pc-tools
```

### 2. 安装前端依赖
```bash
npm install
```

### 3. 安装 Rust 依赖（自动完成）
首次运行 Tauri 时会自动下载并编译 Rust 依赖。

## 三、开发运行

### 启动开发服务器
```bash
npm run tauri dev
```

此命令会同时启动：
- Vue 前端开发服务器 (http://localhost:1420)
- Rust 后端开发环境
- 桌面应用窗口

### 仅启动前端预览（不启动桌面应用）
```bash
npm run dev
```

## 四、构建打包

### 构建前端
```bash
npm run build
```

### 构建桌面应用
```bash
# 开发构建
npm run tauri build

# 生产构建（优化版本）
npm run tauri build -- --release
```

构建产物位置：`src-tauri/target/release/bundle/`

### 预览构建结果
```bash
npm run preview
```

## 五、常用命令汇总

| 命令 | 说明 |
|------|------|
| `npm install` | 安装前端依赖 |
| `npm run dev` | 启动前端开发服务器 |
| `npm run build` | 构建前端生产版本 |
| `npm run preview` | 预览构建结果 |
| `npm run tauri dev` | 启动 Tauri 开发环境 |
| `npm run tauri build` | 构建桌面应用 |
| `npm run tauri build -- --release` | 构建生产版本桌面应用 |

## 六、技术栈说明

| 类型 | 技术 | 版本 |
|------|------|------|
| 前端框架 | Vue | ^3.4.0 |

| 路由管理 | Vue Router | ^4.2.5 |

## 七、注意事项

1. **Windows 环境**: 确保已安装 Visual Studio Build Tools 和 Windows SDK
2. **网络问题**: Rust 依赖下载可能需要配置镜像源
3. **权限问题**: 部分系统信息获取需要管理员权限
4. **首次构建**: 首次构建会下载大量依赖，耗时较长，请耐心等待
